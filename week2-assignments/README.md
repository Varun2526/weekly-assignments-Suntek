# Week 2 — Advanced JavaScript

Advanced JavaScript assignments covering higher-order array functions, ES6 modules, OOP with classes, spread/rest operators, shallow/deep copy, and timer functions.

| `assignment2.js` | `filter()` | Filter employees by department and salary |

| `assignmentadv1.js` | Multiple HOFs | E-commerce product filtering and price calculations |

| `assignmentadv5.js` | Multiple HOFs | Bank transaction analyzer — credits, debits, balance |

## 💡 Concepts Covered

- Higher-Order Functions (`filter`, `map`, `reduce`, `find`, `findIndex`)
- ES6 Modules (named exports/imports)
- Spread & Rest operators
- Shallow vs Deep copy
- Timer functions (`setTimeout`, `setInterval`)
- Object-Oriented Programming (ES6 Classes)
- Encapsulation and state management
