# Week 2 — Advanced JavaScript

Advanced JavaScript assignments covering higher-order array functions, ES6 modules, OOP with classes, spread/rest operators, shallow/deep copy, and timer functions.

## 📋 Assignments

### Day 1 — Array Higher-Order Functions (HOFs)

| File | Topic | Description |
|------|-------|-------------|
| `hof-temperature-analyzer.js` | `filter`, `map`, `reduce` | Daily temperature data — Celsius to Fahrenheit, averages |
| `hof-course-name-processor.js` | `filter`, `map`, `reduce` | Filter and transform course names |
| `hof-student-marks.js` | `filter`, `map`, `reduce` | Student exam results — grace marks, highest mark |
| `hof-ecommerce-cart.js` | Multiple HOFs | E-commerce cart — filter, total, discounts |
| `hof-student-grades.js` | Multiple HOFs | Student grade system — filter, grade, average, search |
| `hof-employee-payroll.js` | Multiple HOFs | Employee payroll — salary, bonuses, department filtering |
| `hof-movie-streaming.js` | Multiple HOFs | Movie platform — genre filter, ratings, search |
| `hof-bank-transactions.js` | Multiple HOFs | Bank transactions — credits, debits, balance |

### Day 1 — ES6 Modules

| File | Topic | Description |
|------|-------|-------------|
| `es6-module-exports.js` | Named Exports | Export data, objects, and strings |
| `es6-module-imports.js` | Named Imports | Import and use named exports |

### Day 2 — Advanced Concepts

| File | Topic | Description |

## 💡 Concepts Covered

- Higher-Order Functions (`filter`, `map`, `reduce`, `find`, `findIndex`)
- ES6 Modules (named exports/imports)
- Spread & Rest operators
- Shallow vs Deep copy
- Timer functions (`setTimeout`, `setInterval`)
- Object-Oriented Programming (ES6 Classes)
- Encapsulation and state management

## ▶️ How to Run

```bash
node hof-temperature-analyzer.js
node hof-ecommerce-cart.js

# For module files (requires ES module support)
node --experimental-modules es6-module-imports.js
```
