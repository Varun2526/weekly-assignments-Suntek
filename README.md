# 📚 ATP Weekly Assignments — Suntek Academy

A comprehensive collection of weekly programming assignments completed during the **Advanced Training Program (ATP)** at Suntek Academy. This repository documents my learning journey from JavaScript fundamentals through full-stack MERN development.

## 🗂️ Repository Structure

| Week | Topic | Tech Stack | Description |
|------|-------|-----------|-------------|
| [Week 1](./week1-assignments/) | JavaScript Basics | JavaScript | Variables, conditionals, loops, arrays, functions |
| [Week 2](./week2-assignments/) | Advanced JavaScript | JavaScript (ES6+) | Array HOFs, modules, OOP, spread/rest, timers |
| [Week 3](./week3-assignments/) | Node.js & Express | Node.js, Express, MongoDB | REST API development with CRUD operations |
| [Week 4](./week4-assignments/) | HTML Fundamentals | HTML5 | Page structure, lists, links, images, tables |
| [Week 5](./week5-assignments/) | CSS & React Intro | HTML, CSS, React, Vite | CSS layouts, flexbox, React components & props |
| [Week 6](./week6-assignment/) | React & MERN | React, Node.js, Express, MongoDB | Hooks, Context API, Zustand, full MERN CRUD app |
| [Week 7](./week7-assignment/) | Capstone Project | MERN Stack, Cloudinary, JWT | Full-stack BlogApp with auth, roles & admin panel |

## 🛠️ Tech Stack

- **Languages**: JavaScript (ES6+), HTML5, CSS3
- **Frontend**: React 19, Vite, React Router, Zustand
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **Auth**: JSON Web Tokens (JWT), bcryptjs
- **Cloud**: Cloudinary (image uploads)
- **Tools**: Git, npm, REST Client

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Varun2526/weekly-assignments-Suntek.git

# Navigate to any assignment
cd weekly-assignments-Suntek/week1-assignments

# For Node.js assignments (week 3+)
npm install
node <filename>.js

# For React assignments (week 5+)
cd <project-folder>
npm install
npm run dev
```
