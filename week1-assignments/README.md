# Week 1 — JavaScript Basics

Foundational JavaScript assignments covering variables, conditionals, loops, arrays, and functions.

## 📋 Assignments

| File | Topic | Description |
|------|-------|-------------|
| `day_1_assignment1.js` | Comparison | Compare two numbers and find the larger |
| `day_1_assignment2.js` | Comparison | Find the maximum among three numbers |
| `day_1_assignment3.js` | Arrays | Calculate the sum of all elements in an array |
| `day_1_assignment4.js` | Arrays | Find the minimum element in an array |
| `day_1_assignment5.js` | Functions | Function to find the biggest of three numbers |
| `day_1_assignment6.js` | Functions | Function to sum all elements of an array |
| `day_1_assignment7.js` | Functions | Search for an element in an array and return its index |
| `day2-employee-marks.js` | Objects & Arrays | Employee records — insert, remove, and modify using splice |

## 💡 Concepts Covered

- Variables (`let`, `const`)
- Conditional statements (`if`, `else if`, `else`)
- Loops (`for`, `for...of`)
- Arrays and array methods (`push`, `splice`, `length`)
- Functions (declaration, parameters, return values)
- Template literals
- Object manipulation

## ▶️ How to Run

```bash
node day_1_assignment1.js
```
