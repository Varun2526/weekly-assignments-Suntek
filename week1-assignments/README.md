# Week 1 — JavaScript Basics

Foundational JavaScript assignments covering variables, conditionals, loops, arrays, and functions.

## 📋 Assignments

### Day 1 — Conditionals & Arrays

| File | Topic | Description |
|------|-------|-------------|
| `compare-two-numbers.js` | Conditionals | Compare two numbers and find the larger |
| `find-max-of-three.js` | Conditionals | Find the maximum among three numbers |
| `array-sum.js` | Arrays | Calculate the sum of all elements in an array |
| `array-min.js` | Arrays | Find the minimum element in an array |
| `function-find-largest.js` | Functions | Function to find the biggest of three numbers |
| `function-array-sum.js` | Functions | Function to sum all elements of an array |
| `array-search.js` | Functions | Search for an element in an array and return its index |

### Day 2 — Objects & Arrays

| File | Topic | Description |
|------|-------|-------------|
| `employee-marks-splice.js` | Objects & Arrays | Employee records — insert, remove, and modify using splice |

## 💡 Concepts Covered

- Variables (`let`, `const`)
- Conditional statements (`if`, `else if`, `else`)
- Loops (`for`, `for...of`)
- Arrays and array methods (`push`, `splice`, `length`)
- Functions (declaration, parameters, return values)
- Template literals
- Object manipulation

## ▶️ How to Run

```bash
node compare-two-numbers.js
node employee-marks-splice.js
```
