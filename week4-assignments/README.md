# Week 4 — HTML Fundamentals

HTML assignments covering basic page structure, text formatting, lists, links, images, and semantic elements.

## 📋 Assignments

| File | Topic | Description |
|------|-------|-------------|
| `paragraphs-and-links.html` | Text & Links | Paragraphs with bold/italic formatting, headings, external links, and unordered lists |
| `nested-lists.html` | Nested Lists | Nested unordered lists for course categories (DSA, Web Dev, etc.) |
| `cat-profile-page.html` | Complete Page | A profile page for "Pochi the Cat" with headings, images, ordered lists, links, and layout |

## 💡 Concepts Covered

- HTML5 document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Text formatting (`<p>`, `<b>`, `<i>`, `<h1>`–`<h3>`)
- Hyperlinks (`<a href>`)
- Lists (ordered `<ol>`, unordered `<ul>`, nested)
- Images (`<img>` with `src`, `alt`, `width`, `height`)
- Divs and basic layout (`<div>`, `float`)
- Horizontal rules (`<hr>`)
- Meta tags and viewport settings

## ▶️ How to View

Open any `.html` file directly in a web browser:

```bash
open paragraphs-and-links.html
```
